# Part I: Text - Prompt Optimization

TextGrad-based prompt optimization for FET sensor information extraction.

## Scripts

- `textgrad_train.py` - Main training script
- `textgrad_evaluate.py` - Test set evaluation

## Prompts

Optimized prompts for 5 field groups (M2/M7/M8 variants + human baseline):
- `core/` - Core sensor fields
- `material_layers/` - Material layer extraction
- `electrodes_architecture/` - Electrode configuration
- `sensitivity_response/` - Sensitivity metrics
- `synthesis_thermal/` - Synthesis conditions
