# Data Descriptor

## Macroscopic Properties

The `macroscopic_properties` field contains 25 sequence-derived biochemical features calculated using BioPython's ProteinAnalysis module. All properties are computed directly from the amino acid sequence without requiring 3D structural information.

### Property Definitions and Units

| Property | Unit | Definition | Range |
|----------|------|------------|-------|
| `sequence_length` | aa (amino acids) | Total number of amino acid residues in the protein sequence | 1 - N |
| `molecular_weight_kDa` | kDa (kiloDaltons) | Theoretical molecular weight calculated from amino acid composition | 0.1 - 1000+ |
| `isoelectric_point` | pH units | pH at which the protein has zero net charge | 3.0 - 12.0 |
| `aromaticity` | Fraction (0-1) | Relative frequency of aromatic amino acids (Phe, Trp, Tyr) | 0.0 - 0.3 |
| `instability_index` | Dimensionless | Estimate of protein stability in vitro (>40 suggests unstable) | 0 - 100+ |
| `gravy` | Dimensionless | Grand Average of Hydropathy; positive = hydrophobic, negative = hydrophilic | -2.0 - +2.0 |
| `charge_at_pH7` | Elementary charges | Net protein charge at pH 7.0 (physiological pH) | -100 - +100 |
| `charge_at_pH5` | Elementary charges | Net protein charge at pH 5.0 (acidic condition) | -100 - +100 |
| `aromatic_fraction` | Percentage (%) | Fraction of aromatic residues (F, W, Y) | 0 - 30 |
| `aliphatic_fraction` | Percentage (%) | Fraction of aliphatic residues (A, V, L, I) | 0 - 50 |
| `polar_fraction` | Percentage (%) | Fraction of polar uncharged residues (S, T, N, Q) | 0 - 40 |
| `charged_fraction` | Percentage (%) | Fraction of charged residues (K, R, D, E) | 0 - 40 |
| `positive_fraction` | Percentage (%) | Fraction of positively charged residues (K, R, H) | 0 - 30 |
| `negative_fraction` | Percentage (%) | Fraction of negatively charged residues (D, E) | 0 - 30 |
| `helix_fraction` | Fraction (0-1) | Predicted propensity for خ±-helix secondary structure | 0.0 - 1.0 |
| `turn_fraction` | Fraction (0-1) | Predicted propensity for خ²-turn secondary structure | 0.0 - 1.0 |
| `sheet_fraction` | Fraction (0-1) | Predicted propensity for خ²-sheet secondary structure | 0.0 - 1.0 |
| `cysteine_count` | Count | Absolute number of cysteine residues (potential disulfide bonds) | 0 - 50+ |
| `proline_count` | Count | Absolute number of proline residues (structure-breaking) | 0 - 100+ |
| `glycine_fraction` | Percentage (%) | Fraction of glycine residues (conformational flexibility) | 0 - 20 |
| `extinction_coefficient_reduced` | Mâپ»آ¹ cmâپ»آ¹ | Molar extinction coefficient at 280 nm (all Cys reduced) | 0 - 200,000+ |
| `extinction_coefficient_oxidized` | Mâپ»آ¹ cmâپ»آ¹ | Molar extinction coefficient at 280 nm (all Cys oxidized as disulfide) | 0 - 200,000+ |
| `average_flexibility` | Dimensionless | Mean flexibility score based on Vihinen scale (higher = more flexible) | 0.9 - 1.1 |
| `tiny_fraction` | Percentage (%) | Fraction of tiny residues (A, C, G, S, T) | 0 - 50 |
| `large_fraction` | Percentage (%) | Fraction of large residues (F, I, K, L, M, R, W, Y) | 0 - 50 |

### Notes on Macroscopic Properties

- **Fractions** are reported as percentages (0-100) or normalized fractions (0.0-1.0) as indicated
- **Secondary structure fractions** are predictions based on the Chou-Fasman algorithm and do not reflect experimental structures
- **Instability index** > 40 suggests the protein may be unstable in a test tube (not necessarily in vivo)
- **GRAVY** values above 0 indicate hydrophobic proteins; below 0 indicates hydrophilic
- **Extinction coefficients** measure protein concentration via UV absorbance at 280 nm, useful for quantification
  - Reduced form: assumes all cysteines are free thiols (reduced state)
  - Oxidized form: assumes all cysteines form disulfide bonds (oxidized state)
  - Based on Trp, Tyr, and Cys content
- **Average flexibility** is computed using the Vihinen flexibility scale; higher values indicate more flexible/disordered regions
- **Tiny/Large fractions** categorize residues by volume; useful for predicting packing and structural compactness
- All properties are calculated on cleaned sequences where non-standard amino acids (U, B, Z, X, J, O) have been replaced with standard approximations

## Protein Embeddings

The `embedding` field contains a 320-dimensional numerical representation of the protein sequence generated by ESM-2 (Evolutionary Scale Modeling version 2), a state-of-the-art protein language model developed by Meta AI. ESM-2 is a transformer-based neural network pre-trained on 65 million protein sequences from UniRef50, enabling it to learn evolutionary patterns and structural relationships encoded in protein sequences. The model processes the amino acid sequence and generates contextual embeddings where each dimension captures latent features related to biochemical properties, evolutionary conservation, structural propensities, and functional motifs. These 320-dimensional vectors serve as dense, information-rich representations of protein sequences that preserve both local (individual residue characteristics) and global (overall sequence context) information, making them suitable for downstream machine learning tasks such as protein classification, function prediction, structure prediction, and similarity analysis. The embeddings are generated using mean pooling over the sequence length from the final hidden layer of the `facebook/esm2_t6_8M_UR50D` model variant, which balances computational efficiency with representation quality. Unlike hand-crafted features, these learned embeddings capture complex, non-linear relationships between amino acids that may not be apparent from traditional biochemical descriptors alone.

### Embedding Specifications

- **Model**: `facebook/esm2_t6_8M_UR50D`
- **Dimensionality**: 320
- **Value range**: Approximately -10.0 to +10.0 (unbounded, model-dependent)
- **Pooling method**: Mean pooling across sequence positions
- **Maximum sequence length**: 1024 amino acids (longer sequences are truncated)
- **Data type**: 32-bit floating point numbers

### Interpretation

- Each of the 320 dimensions represents a learned latent feature
- Dimensions are not directly interpretable (unlike macroscopic properties)
- Cosine similarity or Euclidean distance between embeddings reflects protein sequence similarity
- Embeddings capture evolutionary relationships, structural homology, and functional similarity
- Can be used directly as input features for machine learning models
