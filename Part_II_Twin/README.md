# Part II: Twin - Digital Twin Construction

Heterogeneous graph construction for FET sensors.

## Structure

- `descriptor_query_tools/` - Material descriptor extraction (molecules, polymers, inorganic, biomolecules, DNA/RNA)
- `data_augmentation/` - Physics-aware augmentation and graph building
- `gnn_training/` - DTE-GNN model and baselines
- `sample_data/` - Anonymized sample records
